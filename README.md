# Schumi

## **Macroprogetto: Personalizzazione del Modello di Guida**

Il nostro obiettivo principale è **adattare il modello di guida esistente (model-based) allo stile unico di ciascun utente**.

Per raggiungere questo scopo, ogni persona guiderà e creerà il proprio **dataset personalizzato**. Questi dati verranno poi utilizzati per "allenare" e **specificare il modello di guida** al comportamento individuale, garantendo che le raccomandazioni o le analisi dell'app riflettano fedelmente le abitudini di guida di ciascun conducente.
